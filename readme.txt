== Description ==

A very cool theme inspired by file explorer. Created in <a href="https://shellcreeper.com/48-hours-theme-challenge/">48 Hours Theme Challenge</a>.

Theme features:

* Responsive Layout
* Custom Background
* Custom CSS
* One Navigation Menus (Custom Page Template)
* One Widget Areas (Sidebars, using toggle)
* Post Formats

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How to setup Home Navigation Page Template =

"Home Navigation" page template is a page template created to display "Home Navigation" menu to replace page content. Ideally used in front page as landing page.

You can easily do this to setup this page:
1. Go to "Settings > Reading" and set "A static page" for your "Front Page".
2. In that page, select page template "Home Navigation" from "Page Attributes" Meta Box and save.
3. Go to "Appearance > Menus" and assign a navigation menu to "Home Navigation Page Template" menu location.
